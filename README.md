# ✦ RUNEFALL

ルーンタイルを組み合わせて呪文を発動し、8フロアのダンジョンを攻略する  
スマートフォンブラウザ向けローグライク×パズルゲーム。

🎮 **[今すぐプレイ](https://men0tai0ko.github.io/runefall/)**

---

## 概要

| 項目 | 内容 |
|------|------|
| ジャンル | ローグライク × パズル |
| 対象端末 | スマートフォンブラウザ（PC でも動作） |
| プレイ時間 | 1ラン 約15〜20分 |
| バージョン | v1.7.1 |

---

## 主な機能

- **6属性ルーンタイル × 27種コンボ呪文** — タイルを最大3枚選択してコンボを狙う
- **8フロアランダムダンジョン** — 戦闘 / ショップ / 休憩 / 宝箱 / イベントがランダム配置
- **4段階難易度** — EASY / NORMAL / HARD / VOID（VOID はランク解放）
- **6種デッキプリセット** — 初期デッキを選んでビルドを方針づけ
- **ランクシステム（7段階）＋ 実績（30種）** — プレイ累積で永続強化が解放される
- **スコア保存** — 難易度ごとのベストスコアを localStorage に永続保存

---

## 使い方

ブラウザで URL を開くだけでプレイ可能。インストール・ビルド不要。

```
1. 難易度を選択
2. デッキプリセットを選択
3. マップの部屋をタップして進む
4. タイルを最大3枚選択 → 「詠唱」で呪文発動
5. F8ボス（魔竜）を撃破するとクリア
```

ローカルで開く場合は `index.html` をブラウザにドラッグ＆ドロップ。

## 開発

**更新手順**

```
1. index.html を修正
2. APP_VERSION を更新（例: '1.7.0' → '1.7.1'）
3. コミット・プッシュ
```

バージョン変更だけでキャッシュバスティングが全端末に自動適用されます。

**技術スタック**

| 項目 | 内容 |
|------|------|
| 実装 | HTML / CSS / JavaScript（ES2020+） |
| 外部依存 | Google Fonts のみ（ゲーム動作には不要） |
| 描画 | Canvas API（DPR対応） |
| サウンド | Web Audio API |
| 保存 | localStorage |

詳細は [spec.md](spec.md) を参照。

---

## 注意事項

- `file://` プロトコルでは localStorage が制限される場合があります（GitHub Pages 推奨）
- セッション中のゲーム状態は保存されません（ローグライクの原則）
