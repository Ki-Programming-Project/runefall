# RUNEFALL — ゲームプレイ仕様書

## 概要

RUNEFALLはスマートフォンブラウザ向けのローグライク×パズルゲームです。プレイヤーは属性ルーンタイルを組み合わせて呪文を発動し、8フロアのダンジョンを攻略します。外部ライブラリを使用しない単一HTMLファイル構成で、ファイルをブラウザで開くだけで即プレイ可能です。

---

## ゲームループ

ゲームは以下の順序で進行します。タイトル画面で難易度を選択し、マップ画面でフロアを選択して部屋に入ります。全8フロアの最後にボス部屋（魔竜）が固定配置されており、撃破するとクリアです。プレイヤーのHPが0になった時点でゲームオーバーとなりランが終了します。ローグライクの原則に従い、ゲームオーバー時のセーブ状態引き継ぎは行いません。再挑戦時は前回の難易度を引き継ぎます。

```
タイトル（難易度選択）
  └→ マップ画面
       ├→ 戦闘（enemy / boss）
       │    └→ 勝利: 報酬選択 → マップへ
       │    └→ 敗北: ゲームオーバー
       ├→ ショップ（タイル購入 / ステータス強化）
       ├→ 休憩（HP+30回復）
       ├→ 宝箱（Gold獲得）
       └→ イベント（ランダムイベント）
```

| 難易度 | 初期HP | 手札枚数 | 敵ATK倍率 | 敵HP倍率 | Gold倍率 | デッキ上限 |
|--------|--------|----------|-----------|----------|----------|------------|
| EASY   | 100    | 6        | ×0.7      | ×0.8     | ×1.2     | 無制限     |

## 戦闘システム

### ターン進行

1. プレイヤーがタイルを1〜3枚選択して「詠唱」を押す
2. 呪文効果が適用される（ダメージ / 回復 / 状態異常付与）
3. 敵が反撃行動を行う（画面シェイクあり）
4. 敵のHPが0になるまで繰り返す
5. 「スキップ」ボタンでターンを消費して敵に攻撃させることも可能

### タイル選択ルール

手牌はデッキからランダムに抽選されます（手札サイズ枚）。タイルは1〜3枚まで選択可能で、選択した組み合わせに一致するコンボ呪文があれば発動します。一致しない場合は選択タイルのパワー合計値がそのまま基本攻撃ダメージになります。詠唱後は手牌が新たに引き直されます。

### 演出

コンボ発動時は属性カラーのパーティクルを50粒子、基本攻撃時は20粒子を生成します。敵の反撃時は攻撃種別に応じた強度の画面シェイクが発生します。

### 状態異常

| 効果ID | 内容 | 付与条件 |
|--------|------|---------|
| burn   | 被ダメージ+3（次回攻撃時） | Inferno / Hellfire |
| stun   | 敵の次のターンをスキップ   | Boulder |
| slow   | 敵ATKを70%に軽減           | Storm |
| curse  | 視覚的フィードバックのみ   | Eclipse / Abyss |
| blind  | 視覚的フィードバックのみ   | Steam |
| pierce | 視覚的フィードバックのみ   | Gaia Beam |
| wet    | 少量回復を付与             | Flood |

## マップシステム

ダンジョンは全8フロアで構成されます。F8は常にボス部屋です。F1〜F7はランダムに以下の部屋タイプが配置されます。

| タイプ   | 出現ウェイト | 内容 |
|----------|-------------|------|
| enemy    | 3/7         | 通常戦闘 |
| shop     | 1/7         | タイル購入・ステータス強化 |
| treasure | 1/7         | Gold獲得（20〜39G × goldRate）|
| rest     | 1/7         | HP+30回復 |
| event    | 1/7         | ランダムイベント |

---

## ショップシステム

ショップはタイル購入とステータス強化の2タブで構成されます。タイル購入は15G固定でランダムな属性タイルを4種から選択できます。HARDのデッキ上限（20枚）到達時は購入不可。ステータス強化は購入するたびにコストが10G増加します（初期コスト + 現Lv × 10G）。

### ステータス強化一覧

| 名称       | 初期コスト | 最大Lv | 効果 |
|------------|------------|--------|------|
| 体力強化   | 20G        | Lv3    | 最大HP +20/Lv（難易度基準HP + lv×20）|
| 攻撃強化   | 25G        | Lv3    | 全ダメージ +2/Lv |
| 回復強化   | 20G        | Lv2    | 回復量 +6/Lv |
| 財宝の加護 | 15G        | Lv2    | Gold獲得倍率 +30%/Lv |
| 手札拡張   | 30G        | Lv2    | 手札枚数 +1/Lv（難易度基準手札 + lv）|

## ランダムイベント

| 名称       | アイコン | 内容 |
|------------|----------|------|
| 祝福の泉   | 🌟       | デッキのタイルを1枚選んでパワーを+1強化 |
| 呪いの祭壇 | 💀       | HP-20と引き換えに新タイルを1枚入手 |
| 運命の賭け | 🎲       | 50%でHP全回復、50%でHP半減 |
| 旅の魔術師 | 🧙       | デッキのランダムなタイル1枚を別属性に変換 |

すべてのイベントは「断る」ことが可能です。

## スコアシステム

```
スコア = (到達フロア×100 + ターンボーナス + Goldボーナス + クリアボーナス) × 難易度倍率

ターンボーナス = max(0, 500 - 総ターン数 × 8)
Goldボーナス  = floor(所持Gold × 0.5)
クリアボーナス = クリア時 300、ゲームオーバー時 0

難易度倍率: EASY=0.7 / NORMAL=1.0 / HARD=1.5
```

ベストスコアは難易度ごとにlocalStorageに永続保存され、タイトル画面に表示されます。

## チュートリアル

初回プレイ時のみ、最初の戦闘開始800ms後に5ステップのチュートリアルが起動します。完了後は `runefall_tutorial_done` フラグをlocalStorageに保存し、以降は表示されません。

## ボス特殊行動（魔竜）

| トリガー           | 行動名   | 効果 | シェイク |
|--------------------|----------|------|---------|
| HP40%以下（初回）  | 激怒     | ATKが1.8倍に永続上昇 | 強度8 / 400ms |
| 3ターンごと        | 炎の息吹 | ATK×1.6 + burn付与 | 強度10 / 500ms |
| 5ターンごと        | 竜の咆哮 | 通常ダメージ + 手牌リフレッシュ | 強度6 / 350ms |
| 上記以外           | 通常攻撃 | ATKダメージ | 強度4 / 200ms |
